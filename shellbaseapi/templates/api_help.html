/api/v1/stations

This request returns metadata about the station, such as geographic location and samples available.

Attributes

bbox string
The query will return all stations within the bounding box.

bbox=xmin,ymin,xmax,ymax

Example Query

/api/v1/stations/state abbreviation

This request returns metadata about the station, such as geographic location and samples available. The query will be limited to the stations in the state provided.

The abbreviations are: NC, SC, GA, or FL. If the bbox parameter is provided, the request returns only the stations within the state within the bbox.

Attributes

bbox string
The query will return all stations within the bounding box.

bbox=xmin,ymin,xmax,ymax

Example Query
