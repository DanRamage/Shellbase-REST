Shellbase REST API

/api/v1/metadata/stations

This request returns a list of all the stations. If the BBOX parameter is provided, only the stations within the bounds will be listed.

Attributes

bbox string
The query will return all stations within the bounding box.

bbox=xmin,ymin,xmax,ymax

type string
The format to receive the data, if not provided the default is GeoJSON. Accepted values are csv or json.

type=csv|json

JSON Example Query: http://shellbaseapi.howsthebeach.org/api/v1/metadata/stations

CSV Example Query: http://shellbaseapi.howsthebeach.org/api/v1/metadata/stations?type=csv

JSON Example BBOX Query: http://shellbaseapi.howsthebeach.org/api/v1/metadata/stations?bbox=-81.1,32.8,-77.8,34

CSV Example BBOX Query: http://shellbaseapi.howsthebeach.org/api/v1/metadata/stations?bbox=-81.1,32.8,-77.8,34&type=csv

/api/v1/metadata/stations/state

This request returns a list of all the stations in the given state. If the BBOX parameter is provided, only the stations within the bounds will be listed. The formats for the state field can be: NC, SC, GA, or FL

Attributes

bbox string
The query will return all stations within the bounding box.

bbox=xmin,ymin,xmax,ymax

type string
The format to receive the data, if not provided the default is GeoJSON. Accepted values are csv or json.

type=csv|json

State Specific JSON Example Query: http://shellbaseapi.howsthebeach.org/api/v1/metadata/stations/sc

State Specific CSV Example Query: http://shellbaseapi.howsthebeach.org/api/v1/metadata/stations/sc

/api/v1/metadata/stations/state/station name

This request returns metadata about the specific station, such as geographic location and samples available. The formats for the state field can be: NC, SC, GA, or FL

Attributes

type string
The format to receive the data, if not provided the default is GeoJSON. Accepted values are csv or json.

type=csv|json

JSON Example Query: http://shellbaseapi.howsthebeach.org/api/v1/metadata/stations/SC/19-01

CSV Example Query: http://shellbaseapi.howsthebeach.org/api/v1/metadata/stations/SC/19-01?type=csv

/api/v1/data/state abbreviation/station name

This requests returns data for the provided station. The query contains the start_date and end_date parameters for the time period of interest. The state abbreviations are: NC, SC, GA, or FL.

Attributes

start_date string
The date to begin the data request at.

start_date=YYYY-MM-DD

end_date string
The date to end the data request at.

end_date=YYYY-MM-DD

type string
The format to receive the data, if not provided the default is GeoJSON. Accepted values are csv or json.

type=csv|json

JSON Example Query: http://shellbaseapi.howsthebeach.org/api/v1/data/sc/19-01?start_date=2019-01-01&end_date=2020-01-01

CSV Example Query: http://shellbaseapi.howsthebeach.org/api/v1/data/sc/19-01?start_date=2019-01-01&end_date=2020-01-01&type=csv
