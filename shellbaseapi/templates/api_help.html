/api/v1/stations

This request returns metadata about the station, such as geographic location and samples available.

Attributes

bbox string
The query will return all stations within the bounding box.

bbox=xmin,ymin,xmax,ymax

/api/v1/stations/state abbreviation

This request returns metadata about the station, such as geographic location and samples available. The query will be limited to the stations in the state provided.

The abbreviations are: NC, SC, GA, or FL. If the bbox parameter is provided, the request returns only the stations within the state within the bbox.

Attributes

bbox string
The query will return all stations within the bounding box.

bbox=xmin,ymin,xmax,ymax

/api/v1/data/state abbreviation/station name

This requests returns data for the provided station. The query contains the start_date and end_date parameters for the time period of interest. The state abbreviations are: NC, SC, GA, or FL.

Attributes

start_date string
The date to begin the data request at.

start_date=YYYY-MM-DD

end_date string
The date to end the data request at.

end_date=YYYY-MM-DD

type string
The format to receive the data, if not provided the default is GeoJSON. Accepted values are csv or json.

type=csv|json
